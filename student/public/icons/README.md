# App Icons

Place your app icons in this directory with the following sizes:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Generating Icons

You can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- Figma or Adobe XD with icon export plugins

## Requirements

- PNG format
- Square dimensions
- Transparent or solid background
- Clear, simple design that works at small sizes

## Maskable Icons

For Android adaptive icons, create maskable versions with safe zone padding (80% of canvas).
